# 新增功能：微课视频、AI 动画课程与动态讲题窗口

本次在 ArmEdu 系统上继续完成了“文字微课 + AI 动画课程 + 动态讲题”的一体化扩展，重点不是单独做一个视频页，而是把它们真正接入现有的“测—学—评”闭环。

## 1. 本次新增/完善的能力

### 1.1 智能学习页改版
- 微课目录改为更清晰的一行式目录布局，不再挤成窄列
- 补齐全部 8 个章节的微课入口
- 每章统一提供：
  - 文字微课
  - AI 动画课程
  - 完成记录
  - 当前掌握度展示

### 1.2 微课视频 / AI 动画课程
- 新增 `lesson_videos`、`lesson_video_progress` 数据表
- 启动或 seed 后会自动补齐 8 个章节的演示微课与动画课程
- 动画课程支持：
  - 分镜播放
  - 暂停 / 继续
  - 上一幕 / 下一幕
  - 点击场景跳转
  - 当前幕语音朗读
  - 观看进度回传
  - 标记学习完成
- 管理端支持重新生成动画课程
- 若配置外部 OpenAI-compatible 模型，优先生成 richer storyboard；否则自动回退离线模板

### 1.3 AI 教辅新增“动态讲题窗口”
- 在 AI 教辅页新增独立的动态讲题区域
- 支持输入数学题、物理题后自动生成分镜动画讲解
- 当前支持的典型场景：
  - 傅里叶级数逼近
  - 三角函数图像变化
  - 抛体运动
  - 简谐振动
  - 通用白板推导
- 每个场景会生成：
  - 场景标题
  - 讲解要点
  - narration 旁白
  - equation 公式
  - visual 绘图参数
- 前端使用 Canvas 实时绘制图像和过程，支持语音朗读当前场景

## 2. 新增接口

### 2.1 微课视频相关
- `GET /api/lessons/{skill_id}/video`
- `POST /api/lessons/{skill_id}/video/progress`
- `POST /api/lessons/{skill_id}/video/complete`
- `GET /api/admin/lesson_videos`
- `POST /api/admin/lessons/{skill_id}/video/generate`

### 2.2 动态讲题相关
- `POST /api/ai/visual_solve`

## 3. 运行说明

```bash
python -m pip install -r requirements.txt
python -m scripts.seed
python -m uvicorn app.main:app --reload
```

学员端登录后进入：
- 智能学习 → 微课目录 → 阅读 / 动画
- AI 教辅 → 动态讲题窗口
- 题目解析结果页 → 动态讲题

管理员登录后进入：
- 知识点画像 → AI生成动画课

## 4. 实现说明

### 4.1 关于“动画课程”
本系统实现的是**适合毕业设计落地的 AI 动画课程**：
- 通过脚本/分镜生成教学内容
- 浏览器端 Canvas 负责动态绘制
- 结合字幕、旁白、步骤和图形过程进行讲解
- 可以离线演示

它不是重型云端视频生产平台，但在“课程自动生成 + 动画演示 + 语音讲解 + 进度回流”这一层面已经形成完整功能链路。

### 4.2 关于“动态讲题”
动态讲题重点服务数学、物理题的可视化解释，不只是普通文字回答。系统会把题目转成场景式讲解，边画边讲，更适合答辩展示和教学演示。


## 6) 基于论文实现的 AI 动画制作链路
- 动画课程页新增“制作流程”面板，按论文中的技术路线展示：双智能体分镜 → JSON 中间表示（voiceText / voiceRole / imageDesc）→ Manim 代码草案 → 静态检查 → 运行校验 → 配音字幕与渲染。
- 每一幕都会展示当前的教师/学生讲解轮次和对应的 Manim 代码草案，便于答辩时说明“不是简单轮播，而是可继续落到动画生产”的实现路径。
- 动态讲题窗口也新增了题型识别、分镜规划和绘图脚本面板，方便展示数学/物理题的可视化讲解过程。
